# React Boilerplate

## Packages Included

- react
- react-router-dom
- express
- vite
- vitest
- eslint

## Setup

- [ ] Clone this repo, navigate to it, install packages, and start the server with `npm run dev`
  <details style="padding-left: 2em">
    <summary>Tip</summary>

    ```sh
    npm install
    npm run dev
    ```
  </details>

  - Don't worry about the Dockerfile, and the nginx.conf file. These are only relevant if you wish to deploy this project in the future.
